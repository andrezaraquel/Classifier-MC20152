README

Os valores aceitáveis como argumento para o script classificar.sh são:

1º Parâmetro: "NaiveBayes";
2º Parâmetro: "Caminho do diretório de arquivos de treino";
3º Parâmetro: "Caminho do diretório de arquivos de teste";

Para o 1º argumento, a técnica que deve ser utilizada é a NaiveBayes.

Exemplo de execução:
 
./classificar "NaiveBayes" "/home/user/metodologia/dataSet/treino" "/home/user/metodologia/dataSet/teste" > out.txt
